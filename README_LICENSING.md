## License Key Format

Your license key will look like: `SWFT-P26A-7K9M-X4R2`

- **SWFT** - SwiftMentor prefix
- **P** - Personal license (C = Commercial, D = Developer)
- **26A** - Year code
- **7K9M-X4R2** - Unique identifier
