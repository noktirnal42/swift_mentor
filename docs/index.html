The built-in playground executes real Swift code using the system /usr/bin/swift interpreter. No sandbox, no waiting — write, run, and iterate at the speed of thought.

✓ Real Swift execution via /usr/bin/swift
✓ WWDC18-inspired syntax highlighting powered by Splash
✓ Smart-quote prevention — your code always compiles
✓ Snippet quick-insert bar for common patterns
✓ Execution history with one-click replay
