## What You Get

### 24 Learning Paths, 124 Lessons

SwiftMentor now covers the full Apple platform stack:

- **Swift Foundations**: Swift Fundamentals, Advanced Swift, Swift Concurrency, Protocol-Oriented Programming
- **App Architecture & Data**: App Architecture Patterns, Data Persistence, SwiftData Essentials, Networking & APIs
- **Platform Development**: iOS, macOS, watchOS, tvOS, and visionOS
- **Framework Depth**: SwiftUI, WidgetKit, Combine, Notifications, Core Location & MapKit, AVFoundation & Media, Core ML & Vision, Metal
- **Professional Workflow**: Testing & QA, Xcode Mastery, App Store Distribution

### Live Code Playground

- Execute Swift snippets locally with the system Swift interpreter
- Use syntax highlighting, snippet quick insert, and execution history
- Practice lesson concepts without leaving the app

### Progress & Practice

- Track completion across every path and lesson
- Reinforce concepts with quizzes and exercises
- Build consistency with streaks, achievements, and a dashboard view

### Desktop-First Experience

- Native macOS interface
- Search across lesson content
- Settings, onboarding, snippets, and playground tools designed for repeated study sessions
